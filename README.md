## 开普勒三定律

### 开普勒第一定律

- 所有行星绕太阳的轨道都是椭圆，太阳在椭圆的一个焦点上。

### 开普勒第二定律

- 行星和太阳的连线在相等的时间间隔内扫过的面积相等。

### 开普勒第三定律

- a：半轴长 | T：周期 | k为常数

- $$
  \frac{a^3}{T^2}=k
  $$

## 实验原理

- 半长轴近似成半径
- 利用python拟合
  - scipy库里的optimize下的curve_fit函数进行拟合
- 利用拟合出的矢量图找到多组数据验证定律
